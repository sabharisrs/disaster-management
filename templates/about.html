About

Built to make emergency coordination simpler.

This platform is designed to help communities respond faster during emergencies. Victims can create requests with accurate map locations, volunteers can accept assignments, and administrators can monitor platform activity.

The application is beginner-friendly and built with Flask, SQLite, Bootstrap 5, and OpenStreetMap via Leaflet.

What makes it useful

Clear request statuses keep everyone aligned.

Map-based location sharing improves response time.

Role-based dashboards reduce confusion for each user type.
